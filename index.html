Struktura stranice

HTML fajl sadrži praznu tabelu sa id atributima. JavaScript koristi te ID-ove da bi znao gde da ubaci redove.

thead#zaglavlje-tabele — ovde JS ubacuje <th> naslove kolona
tbody#telo-tabele — ovde JS ubacuje <tr> redove sa podacima
<script> na kraju body-ja — važno! HTML mora biti učitan pre JS-a

script.js

Logika — korak po korak

Ovo je "mozak" projekta. Radi 6 stvari: učitava, parsira, razdvaja header od podataka, sortira, gradi DOM, i ubacuje u tabelu.

script.js — kompletan kod 
fetch('podaci.csv')
  .then(odgovor => odgovor.text())
  .then(tekst => {

    // 1. Razdvoji tekst na redove
    const redovi = tekst.trim().split('\n');

    // 2. Zaglavlje (prvi red)
    const zaglavlje = redovi[0].split(',');
    const trZaglavlje = document.createElement('tr');
    zaglavlje.forEach(naziv => {
      const th = document.createElement('th');
      th.textContent = naziv;
      trZaglavlje.appendChild(th);
    });
    document.getElementById('zaglavlje-tabele').appendChild(trZaglavlje);

    // 3. Podaci (svi redovi osim prvog)
    const podaci = redovi.slice(1).map(red => red.split(','));

    // 4. Sortiraj po 3. koloni (Razred, index 2)
    podaci.sort((a, b) => Number(a[2]) - Number(b[2]));

    // 5. Ubaci redove u tabelu
    const telo = document.getElementById('telo-tabele');
    podaci.forEach(kolone => {
      const tr = document.createElement('tr');
      kolone.forEach(vrednost => {
        const td = document.createElement('td');
        td.textContent = vrednost;
        tr.appendChild(td);
      });
      telo.appendChild(tr);
    });

  });

Objašnjenje svakog dela

fetch('podaci.csv') — šalje HTTP zahtev za fajl. Vraća Promise.
.then(odgovor => odgovor.text()) — konvertuje odgovor u čisti tekst.
tekst.trim().split('\n') — uklanja praznine na krajevima, pa deli tekst po novim redovima. Rezultat je niz stringova.
redovi[0].split(',') — uzima prvi red (zaglavlje) i deli ga po zarezima → ['Prezime','Ime','Razred',...]
redovi.slice(1) — preskače prvi red (zaglavlje), uzima sve ostale.
.map(red => red.split(',')) — svaki red pretvara u niz vrednosti → [['Marković','Ana','1','1','4.85'], ...]
Number(a[2]) - Number(b[2]) — sortiranje po indexu 2 (Razred). Number() pretvara string u broj — bez toga "10" bi bio manji od "2".
document.createElement('tr'/'td'/'th') — kreira HTML elemente programski. appendChild() ih ubacuje jedan u drugi.
styles.css

Stilovi tabele

Samo dva pravila — sve što je potrebno da tabela izgleda uredno sa vidljivim linijama.

styles.css 
table {
  border-collapse: collapse;
}

th, td {
  border: 1px solid black;
}
border-collapse: collapse — spaja dvostruke linije između ćelija u jednu. Bez ovoga svaka ćelija bi imala svoj odvojen okvir.
border: 1px solid black — dodaje crnu liniju debljine 1px oko svakog <th> i <td> elementa.

Podaci o učenicima
